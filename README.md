# dart-counter-vue

Website: [Darts Counter](https://dart-counter-ac001.firebaseapp.com)

## Deploy Steps
```
yarn build
firebase deploy
```
If firebase can not connect to project try:
```
firebase logout
firebase login
```
## Project setup
```
yarn install
```
